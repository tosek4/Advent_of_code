# 🎄 Advent of Code

## About

Solutions for yearly Advent of Code challenges.

Advent of Code is a yearly programming event containing daily puzzles that focus on:

- Algorithm design
- Data parsing
- Optimization
- Simulation
- Graph algorithms
- Mathematical reasoning

## Structure

advent-of-code/
├── 2024/
└── 2025/

Each day contains:
dayXX/
│
├── data
    |
    ├── data.txt
    ├── testData.txt
├── day-x.ts
├── day-plus-x.ts
└── readData.ts
